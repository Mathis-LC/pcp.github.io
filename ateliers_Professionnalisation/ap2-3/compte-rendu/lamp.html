Question

Expliquez chacune de vos décisions.

Enter current password for root (enter for none):

None

On utilise none car, pour l’instant, le compte root n’a pas de mot de passe.

Switch to unix_socket authentication [Y/n]

Y

On bascule sur l’authentification unix_socket afin de renforcer la sécurité.

Change the root password? [Y/n]

Y

On définit un mot de passe pour le compte root afin d’améliorer la sécurité, puisqu’il n’en a pas actuellement.

New password:

stssio

On utilise un mot de passe facile à retenir, car l’environnement LAMP mis en place ne sera pas mis en production.

Re-enter new password:

stssio

On remet le mot de passe indiqué plus haut.

Remove anonymous users? [Y/n]

Y

On supprime les utilisateurs anonymes afin de garantir une traçabilité complète et d’identifier clairement chaque utilisateur accédant à la base de données.

Disallow root login remotely? [Y/n]

Y

On désactive la possibilité pour le compte root de se connecter à distance sur la machine.

Remove test database and access to it? [Y/n]

Y

On supprime la base de données test afin d’éviter qu’elle ne soit utilisée par erreur ou de manière non souhaitée.

Reload privilege tables now? [Y/n]

Y

On met maintenant à jour la table des privilèges pour que toutes les options modifiées en répondant aux questions soient prises en compte.

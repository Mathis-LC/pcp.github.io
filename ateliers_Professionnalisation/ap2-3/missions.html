M1: Mise en production sous différents environnement

Vous devez mettre en place 3 environnements:

  1. LAMP: Linux (US24), Apache2, Mysql, PHP
  2. LEPP: Linux (US24), Nginx, PostgreSQL, PHP
  3. LCIP: Debian 13, Caddy, SQLite, PHP

Les environnements sont à implémenter dans l'ordre précédent.

Pour chaque environnement Il faut documenter et recetter:

  1. Mise en place de l'environnement et de la configuration réseau

  2. Validation d'accès à un serveur web: Page statique

  3. Validation d'accès à un serveur web: Page dynamique (PHP)

  4. Validation d'accès à un serveur web : Page dynamique (php) et accès à une base de données

Les machines peuvent être installées sous virtualbox ou sur la ferme.

⚠️ Remarque sur l'addressage IP si vous laissez l’identification DHCP par défaut, vous risquez de recevoir la même adresse IP qu'une autre machine utilisant la même ova, il faut donc, avant le démarrage de la MV, effectuer un changement de d’adresse Mac dans Virtualbox, puis modifier la configuration réseau de la MV :

  • dhcp4: true
  • dhcp-identifier: mac

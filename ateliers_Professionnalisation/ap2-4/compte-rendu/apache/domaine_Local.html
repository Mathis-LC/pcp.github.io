Sécurisation des répertoires des sites par défaut :

  • Pas de possibilité d'affichage du contenu des répertoires
  • Pages par défaut : accueil.html, index.html, index.php
  • Possibilité de sous-traiter toutes les configurations directement dans le répertoire de publication

Branche 2.0


                <VirtualHost *:80>
                    ServerName applifrais.webvislab.local

                    ServerAdmin webmaster@localhost
                    DocumentRoot /var/www/Applifrais/2.0/www
                    DirectoryIndex accueil.html index.html index.php

                    # Logs
                    ErrorLog ${APACHE_LOG_DIR}/error.log
                    CustomLog ${APACHE_LOG_DIR}/access.log combined

                    # Bonne pratiques de sécurité
                    <Directory /var/www/Applifrais/2.0/www>
                        # Désactive:
                        #   ExecCGI: l'exécution de scripts
                        #   Includes: l'inclusion de fichiers
                        #   Indexes: l'affichage du contenu des répertoires
                        Options -ExecCGI -Includes -Indexes

                        # Active les fichiers .htaccess
                        AllowOverride All 

                        # Autorise l'accès au contenu
                        Require all granted
                    </Directory>
                </VirtualHost>

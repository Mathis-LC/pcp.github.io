Tuto Gitlab clé SSH


                    Lancer git-bash
                    Créer une clé SSH: ssh-keygen -o. (appuyer sur entree pour chaque question, ne pas mettre de mot de passe)
                    Afficher votre clé publique: cat ~/.ssh/id_ed25519.pub
                    Ajouter cette clé dans gitlab:
                    Dans la barre supérieure, dans le coin supérieur droit, sélectionnez votre avatar.
                    Sélectionnez "Préférences".
                    Dans la barre latérale de gauche, sélectionnez "Clés SSH".
                    Cliquez sur Ajouter une clé.
                    Dans le champ « Clé », collez votre clé publique.
                    Tester votre identification :

                    $> ssh -T git@[url du serveur gitlab]
                    Welcome to Gitlab, @[utilisateur]!

Configuration de la VM Ubuntu Desktop 24

Ouvrir VirtualBox

Créer une nouvelle machine virtuelle

Pour créer une nouvelle machine virtuelle on cliquera sur le bouton en haut à gauche

Nous allons respecter les règles de nommage imposé par GSB

Nous allons donc nommer la machine virtuelle comme suit :

NomHote_NomSE où NomSE={US24, UD24, W11}, NomHote : voir schéma réseau

La machine virtuelle que nous allons installer étant la VM Ubuntu Desktop 24 nous allons donc la nommer PC01S300_UD24

Nous devons ensuite stocker la VM dans le répertoire qui convient, encore une fois nous allons nous référerai à la documentation fournie

Nous allons donc stocker la VM en suivant le chemin suivant : D:\VMs\SIO1\mathislc\AP2-4-InfraDevWeb\PC01S300_UD24

On importe ensuite le fichier iso de Ubuntu Desktop 24 fourni

Si le fichier est bien fonctionnelle VirtualBox devrait reconnaitre automatiquement de quel système d’exploitation il s’agit

On décochera l’option pour l’unattended installation

Nous allons vérifier que les premiers paramètres sont bien configurés

Nous passerons l’étape de configuration de l’unattented installation

Nous allons configurés la partie hardware de la machine virtuelle

La configuration par défaut proposé par VirtualBox étant différente de celle recommandé par Ubuntu, nous allons modifier ces paramètres pour qu’ils correspondent aux paramètres de la configuration minimale recommandé par Ubuntu.

Nous passerons ensuite à la configuration du disque dur de la machine virtuelle

Nous allons suivre la documentation fournie ou il y est indiqué :

Les VM sont créées avec des disques durs (DD) en mode dynamique, au format vmdk

Nous allons donc suivra la documentation est adapté la taille du disque dur virtuelle en fonction de la configuration minimale recommandé pour Ubuntu Desktop 24

C’est-à-dire que le disque devra faire au moins une taille de 50Go

La configuration par défaut de VirtualBox propose de mettre un disque de 25Go, mais nous allons suivre la configuration minimale recommandé par Ubuntu

Nous ajusterons donc la taille à 50Go pour éviter d’être limiter en place et pour suivre la configuration minimale indiqué par Canonical sur leur site

Nous changerons le type de disque pour être conforme aux recommandations d'Ubuntu

Nous passerons le disque en format VMDK

Une fois la configuration de la VM nous pourrons cliquer sur le bouton finish

Après avoir cliqué sur le bouton finish la VM devrai se créer

Normalement la VM devrait s’ajouter au groupe : AP2-4-InfraDevWeb

Si ce n’est pas le cas on peut l’ajouter manuellement en la déplaçant en maintenant le clic droit de la souris ou en faisant clic droit sur la VM

Et puis on passera notre curseur sur l’option move to group pour la déplacer dans le groupe AP2-4-InfraDevWeb si tenté que celui-ci existe

Si celui-ci n’existe pas on peut toujours le créer en cliquant sur l’option : new group

Le mode de réseau par défaut étant le NAT nous n’avons pas besoin de modifier la configuration de la VM

Nous pouvons maintenant démarrer la VM

Il existe plusieurs manières de démarrer une VM sur VirtualBox : On peut double cliquer sur la VM, On peut faire un clic droit sur la VM puis dans les options qui s’affiche cliquer sur démarrer ou bien on peut cliquer sur le bouton démarrer qui se situe en haut à gauche de l’interface, cependant il faudra veiller à ce que la VM sélectionné soit la bonne.

Nous allons donc démarrer la VM avec l’une des options citées plutôt

Configuration IP du poste PC01S300

Une fois la VM installé et démarré il nous reste à configurer les paramètres réseaux de celle-ci

Pour effectuer cela nous allons nous référer au schéma présent dans la documentation, ainsi qu’a tableau MV Description ou nous avons rempli les informations nécessaires

Il nous faudra appliquer les changements de configuration réseau suivants :

Adresse IP :

172.18.0.1 / 255.255.0.0

Passerelle :

172.18.255.254 / 255.255.0.0

Serveur DNS :

172.18.0.30

Pour changer cela nous allons utiliser l’application terminal

Une fois arrivé sur ce menu on voit que l’application terminal s’y trouve
